# Get list of countries supported by OpenHolidays

Get list of countries supported by OpenHolidays

## Usage

``` r
oh_countries(language = "EN")
```

## Arguments

- language:

  Language code (default: "EN").

## Value

A data frame of supported countries.

## Examples

``` r
if (FALSE) { # interactive()
  oh_countries(language = "DE")
}
```
